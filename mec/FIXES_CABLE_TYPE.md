# 🔧 Исправление: Сохранение типа кабеля

## Проблема

❌ При смене типа кабеля в выпадающем списке, тип **не сохранялся** в структуре кабеля  
❌ При повторном открытии кабеля, предыдущий выбор был **потерян**

---

## Что было исправлено

### 1. Функция `_on_cable_change()`

**Было:**
```python
def _on_cable_change(self, edge, v_cable):
    cable_type = v_cable.get()
    if cable_type == 'Користувацький':
        return  # ← ничего не делается!
    # ... расчёт параметров
```

**Стало:**
```python
def _on_cable_change(self, edge, v_cable):
    cable_type = v_cable.get()
    if cable_type == 'Користувацький':
        edge['cable_type'] = cable_type  # ← сохраняем!
        self.on_change()
        return
    # Одразу зберігаємо тип кабелю
    edge['cable_type'] = cable_type  # ← КЛЮЧЕВОЕ: сохраняем ТИП
    # ... расчёт параметров
```

### 2. Восстановление выбранного типа при открытии

**Было:**
```python
v_cable = tk.StringVar(value=e.get('cable_type') or cable_types[0])
```

**Стало:**
```python
# Отримуємо збережений тип або перший з списку
current_cable = e.get('cable_type')
if not current_cable or current_cable not in cable_types:
    current_cable = cable_types[0]
v_cable = tk.StringVar(value=current_cable)  # ← более надёжно
```

## Как теперь работает

### Сценарий 1: Смена типа кабеля

```
1. Пользователь кликает на кабель в интерфейсе
   ↓
2. Правая панель показывает текущий тип (например, "ШВВП 2×1.5")
   ↓
3. Пользователь выбирает новый тип из комбобокса (например, "ШВВП 2×2.5")
   ↓
4. ✓ Тип СОХРАНЯЕТСЯ в структуру кабеля
   ↓
5. Параметры пересчитываются и заполняются
```

### Сценарий 2: Сохранение и загрузка схемы

```
1. Нарисовал схему с кабелями типа "ШВВП 2×1.5" (медь)
   ↓
2. Нажимаю "💾 Схема" → выбираю имя файла (например, "my_network.json")
   ↓
3. ✓ Всё сохраняется: типы кабелей, материалы, параметры
   ↓
4. Закрываю программу
   ↓
5. Открываю программу заново
   ↓
6. Нажимаю "📂 Відкрити" → выбираю "my_network.json"
   ↓
7. ✓ Схема полностью восстановлена со ВСЕМИ типами кабелей!
```

## Структура сохранённой схемы

Когда вы нажимаете "💾 Схема", сохраняется JSON файл типа:

```json
{
  "nodes": {
    "0": {"type": "plc_tx", "x": 100, "y": 100, "name": "TX", "Z": 75},
    "1": {"type": "plc_rx", "x": 300, "y": 100, "name": "RX", "Z": 75}
  },
  "edges": {
    "0": {
      "n1": 0,
      "n2": 1,
      "L": 30,
      "alpha": 144.191599,
      "beta": 1087.281047,
      "Z0": 66.85,
      "cable_type": "ШВВП 2×1.5",     ← ВОТ! Сохраняется
      "material": "copper",            ← И вот!
      "frequency_mhz": 24.0
    }
  },
  "_nid": 2,
  "_eid": 1
}
```
