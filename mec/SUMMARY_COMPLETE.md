## 📝 Главное исправление v1.1

**Что было:**
```python
def _on_cable_change(self, edge, v_cable):
    cable_type = v_cable.get()
    if cable_type == 'Користувацький':
        return  # ← ничего не сохраняется!
```

**Что стало:**
```python
def _on_cable_change(self, edge, v_cable):
    cable_type = v_cable.get()
    if cable_type == 'Користувацький':
        edge['cable_type'] = cable_type  # ← ключевое исправление!
        return
    edge['cable_type'] = cable_type  # ← сохраняем ДО расчёта
    self._on_material_change(edge, e)
```

**Результат:** ✅ Тип кабеля теперь сохраняется надёжно

## 💡 Пояснение архитектуры

### Что такое CABLE_TYPES?
- **В коде** — словарь Python с геометрическими константами
- **Примеры:** диаметр дротов, количество дротов, толщина изоляции
- **Функция:** входные данные для формул расчётов

### Что такое cable_params.json?
- **На диске** — файл JSON с результатами расчётов
- **Примеры:** α=144.19, β=1087.28, Z₀=66.85
- **Функция:** кэш для быстрого доступа без пересчёта

**Разница:** Одно — физика (в памяти), другое — результаты (на диске)
