## 🔧 Исправления

### Исправление 1: Сохранение типа кабеля (v1.1)

| Параметр | Детали |
|----------|--------|
| **Проблема** | Тип кабеля теряется при смене в dropdown |
| **Причина** | `_on_cable_change()` не сохранял значение |
| **Решение** | Добавлено немедленное сохранение `edge['cable_type']` |
| **Файл** | [FIXES_CABLE_TYPE.md](FIXES_CABLE_TYPE.md) |
| **Статус** | ✅ ИСПРАВЛЕНО |

### Исправление 2: Ошибки переполнения в gyperbolic функциях (v1.2)

| Параметр | Детали |
|----------|--------|
| **Проблема** | RuntimeWarning: overflow в `np.cosh()`, `np.sinh()` |
| **Причина** | Гиперболические функции переполняются при больших аргументах |
| **Решение** | Добавлена `safe_hyperbolic()` с асимптотическими приближениями |
| **Файл** | [OVERFLOW_FIX.md](OVERFLOW_FIX.md) |
| **Тест** | [test_overflow_fix.py](test_overflow_fix.py) ✅ ПРОЙДЕНЫ |
| **Статус** | ✅ ИСПРАВЛЕНО |
