## 🎯 ГЛАВНОЕ ИСПРАВЛЕНИЕ v1.1

### Проблема
Тип кабеля не сохранялся при смене в выпадающем списке. При повторном открытии кабеля вместо сохранённого типа показывался первый элемент списка.

### Корень проблемы
В функции `_on_cable_change()` тип кабеля не сохранялся при выборе пользователя. Сохранение происходило только при смене материала.

### Решение
```python
# БЫЛО:
if cable_type == 'Користувацький':
    return  # ничего не сохраняется!

# СТАЛО:
if cable_type == 'Користувацький':
    edge['cable_type'] = cable_type  # ← ключевое исправление!
    return
edge['cable_type'] = cable_type  # ← сохраняем сразу
```

### Результат
✓ Тип кабеля теперь сохраняется надёжно при любой смене
