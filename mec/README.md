# 🎯 Unified MBE Cable Calculator

**Объединённое приложение для расчёта параметров кабельных линий передачи и редактирования сетевых топологий**

---

## 📌 Краткое описание

Это приложение объединяет две программы в одну систему:
1. **Cable Calculator** — Расчёт первичных и вторичных параметров кабелей
2. **Network Scheme Editor** — Визуальное редактирование топологии сети

**Ключевая особенность:** При выборе типа кабеля параметры (α, β, Z₀) **автоматически рассчитываются** и сохраняются!

---

## ✨ Основные возможности

### 🔌 Расчёт кабельных параметров

- ✅ 3 предустановленных типа кабелей (ШВВП 2×1.5, 2×2.5, 2×4)
- ✅ 2 материала жил (медь, алюминий)
- ✅ Автоматический расчёт на стандартной частоте 24 МГц
- ✅ Вычисление:
  - **α** — коэффициент затухания [Нп/м]
  - **β** — коэффициент фазы [рад/м]
  - **Z₀** — волновой импеданс [Ом]

### 🗺️ Редактор сетевой топологии

- ✅ Создание узлов сети (Transmitter, Receiver, Node, Load)
- ✅ Соединение узлов кабелями с автоматическим расчётом параметров
- ✅ Редактирование свойств узлов (импеданс, имя)
- ✅ Редактирование свойств кабелей (тип, материал, длина)

### 💾 Сохранение и загрузка

- ✅ Сохранение схемы в JSON с **полной сохранностью** типов кабелей и материалов
- ✅ Загрузка схемы из JSON с **полным восстановлением** всех параметров
- ✅ Автоматическое кэширование параметров в `cable_params.json`

### 📊 Анализ и построение графиков

- ✅ Каскадный расчёт для сложных сетей
- ✅ Построение графиков зависимости затухания и фазы от частоты
- ✅ Визуализация характеристического импеданса

## 🚀 Быстрый старт

### Запуск программы
```bash

```

### Создание простой схемы

```
1. Нажать "⊡ Приклад" → загружается пример с 2 узлами и кабелем

   - Материал: "copper"

4. ✓ Параметры автоматически заполнены:
   - alpha: 144.19 Нп/м

```

```
1. Нажать "📂 Відкрити"
2. Выбрать ранее сохранённый файл (например, "my_network.json")
```

## 🎓 Основные концепции

### CABLE_TYPES vs cable_params.json

**Путаница:** "Зачем два JSON?"

**Ответ:** Это **не два JSON файла**, а:

- **CABLE_TYPES** = константы в коде (геометрия кабелей)
  - Хранится: в памяти программы (словарь Python)
  - Содержит: диаметры дротов, количество дротов, изоляцию
  - Функция: входные данные для расчётов

- **cable_params.json** = результаты расчётов на диске
  - Хранится: файл на диске (`mec/cable_params.json`)
  - Содержит: α, β, Z₀ для каждой комбинации (кабель + материал + частота)
  - Функция: кэш для избежания пересчёта

📖 Смотрите [CABLE_TYPES_vs_JSON.md](CABLE_TYPES_vs_JSON.md) для подробного объяснения

## 🔧 Расширение функциональности

### Добавление нового типа кабеля

1. Отредактируйте `unified_app.py`
2. Найдите класс `CableCalculator`
3. В словаре `CABLE_TYPES` добавьте новый тип:

```python
CABLE_TYPES = {
    'ШВВП 3×2.5': {              # Новый тип
        'n_wires':    60.0,
        'd_wire_mm':  0.238,
        'Liz1_mm':    0.5,
        'lengthk1_mm':7.0,
        'd1_mm':      2.1,
        'a1_mm':      3.5,
        'r1_mm':      1.05,
        'psi':        0.6,
    },
    # ... остальные
}
```

4. Перезагрузите программу
5. При выборе нового типа произойдёт автоматический расчёт

## 🐛 Решение проблем

### Проблема: "Не вижу выпадающего списка типов кабелей"

**Решение:**
1. Кликните на **кабель** (линию между узлами), не на узел
2. Проверьте, что вы кликнули на кабель, а не в пустое место

### Проблема: "Параметры не рассчитываются"

**Решение:**
1. Убедитесь, что выбран кабель из списка (не "Користувацький")
2. Нажмите на материал (copper или aluminium)
3. Параметры должны рассчитаться автоматически

### Проблема: "Тип кабеля не сохраняется"

**Решение:**
1. Эта проблема уже исправлена в v1.1
2. Убедитесь, что вы используете **последнюю версию** unified_app.py
3. Смотрите [FIXES_CABLE_TYPE.md](FIXES_CABLE_TYPE.md)

## 🎯 Основные формулы

Приложение использует теорию передающих линий:

### Первичные параметры линии

- **R₁** — сопротивление [Ом/м]
- **L₁** — индуктивность [H/м]
- **C₁** — ёмкость [F/м]
- **G₁** — проводимость [S/м]

### Вторичные параметры

- **α = Re(γ)** — коэффициент затухания [Нп/м]
- **β = Im(γ)** — коэффициент фазы [рад/м]
- **Z₀ = √((R₁+jωL₁)/(G₁+jωC₁))** — волновой импеданс [Ом]

где **γ** — постоянная распространения
