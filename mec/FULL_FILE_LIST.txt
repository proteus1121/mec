💡 ВАЖНЫЕ КОНЦЕПЦИИ
================================================================================

CABLE_TYPES (в коде):
  ├─ Физические параметры кабелей
  ├─ Диаметры дротов, количество дротов, изоляция
  ├─ Входные данные для расчётов
  └─ Не меняется во время работы

cable_params.json (на диске):
  ├─ Результаты расчётов (α, β, Z₀)
  ├─ Кэш для ускорения
  ├─ Авто-генерируется при первом расчёте
  └─ Растёт по мере использования

Разница объяснена в: CABLE_TYPES_vs_JSON.md
