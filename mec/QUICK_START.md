# 🚀 Швидкий старт | Quick Reference

**Проста інструкція для розпочаткових користувачів**

---

## 🎯 Головні кроки

### 1️⃣ Запуск
```bash
python unified_app.py
```

### 2️⃣ Завантаження прикладу
```
Натисніть "⊡ Приклад"
→ З'явиться схема з двома вузлами (TX і RX) та кабелем
```

### 3️⃣ Вибір типу кабелю
```
Клікніть на кабель (лінію)
Праворуч побачите комбобокс з типами:
├── ШВВП 2×1.5
├── ШВВП 2×2.5
├── ШВВП 2×4
└── Користувацький

Виберіть один → параметри розрахуються автоматично! ✓
```

### 4️⃣ Вибір матеріалу
```
У тому ж вікні виберіть матеріал:
├── copper (медь)
└── aluminium (алюміній)

Параметри оновляться знову! ✓
```

### 5️⃣ Збереження схеми
```
Натисніть "💾 Схема"
Виберіть файл для збереження (наприклад, "mojaset.json")
✓ Готово! Схема збережена
```

## 🎮 Основні кнопки

| Кнопка | Функція |
|--------|---------|
| **⊡ Приклад** | Завантажити готовий приклад схеми |
| **🗑 Очистити** | Вичистити область малювання |
| **💾 Схема** | Зберегти схему у JSON файл |
| **📂 Відкрити** | Завантажити схему з JSON файлу |
| **✏ Редагувати** | Редагувати вибраний елемент |
| **🗑 Видалити** | Видалити вибраний елемент |
| **▶ РОЗРАХУВАТИ** | Розрахувати каскадні параметри |
| **📈 Графіки** | Показати графіки залежності параметрів від частоти |

## 🔌 Типи вузлів

```
Праве меню → вибір типу вузла:

PLC TX (Передавач)
├── Джерело сигналу
└── Z вхідний: 75 Ом

PLC RX (Приймач)
├── Точка вимірювання
└── Z вхідний: 75 Ом

Node (Проміжний вузол)
├── Точка розгалуження
└── Z: 75 Ом

Load (Навантаження)
├── Резистивне навантаження
└── Z: Слід встановити вручну
```

## 📊 Типи кабелів і їх параметри

### ШВВП 2×1.5 (мідь)
```
Частота: 24 МГц
α (затухання): 144.19 Нп/м
β (фаза): 1087.28 рад/м
Z₀ (імпеданс): 66.85 Ом

### ШВВП 2×1.5 (алюміній)

α (затухання): 238.53 Нп/м  ← більше, ніж у міді
β (фаза): 1087.28 рад/м      ← однакова
Z₀ (імпеданс): 66.85 Ом      ← однакова
```

## 💾 Робота зі схемами

### Збереження схеми
```
1. Натисніть "💾 Схема"
2. Виберіть місце для збереження
3. Введіть ім'я файлу (наприклад: network1.json)
4. Натисніть "Зберегти"
   ✓ Схема збережена!
```

### Завантаження схеми
```
1. Натисніть "📂 Відкрити"
2. Виберіть ваш JSON файл
3. Натисніть "Відкрити"
   ✓ Схема завантажена із всіма параметрами!
```

### Що зберігається
```json
{
  "вузли": ["TX", "RX", "Node", ...],
  "кабелі": [
    {
      "тип": "ШВВП 2×1.5",
      "матеріал": "copper",
      "довжина": 30,
      "α": 144.19,
      "β": 1087.28,
      "Z₀": 66.85
    }
  ],
  "...": "все параметри"
}
```

## 🎨 Редагування елементів

### Редагувати кабель
```
1. Клікніть на кабель (лінія)
2. Натисніть "✏ Редагувати"
3. Змініть:
   ├── Тип кабелю
   ├── Матеріал
   └── Довжину
4. Параметри оновляться
```

### Видалити кабель
```
1. Клікніть на кабель
2. Натисніть "🗑 Видалити"
   ✓ Кабель видалений
```

### Редагувати вузол
```
1. Клікніть на вузол (точка)
2. Натисніть "✏ Редагувати"
3. Змініть:
   ├── Тип вузла (TX, RX, Node, Load)
   ├── Імпеданс Z
   └── Ім'я
4. Властивості оновляться
```

## 📈 Розрахунки

### Простий розрахунок (один кабель)
```
1. Натисніть "⊡ Приклад"
2. Виберіть тип кабелю та матеріал
   → Параметри α, β, Z₀ заповнені автоматично!
3. Все готово до розрахунку ✓
```

### Каскадний розрахунок (кілька кабелів)
```
1. Побудуйте схему: TX ─[кабель]─ RX
2. Натисніть "▶ РОЗРАХУВАТИ"
3. Результат:
   ├── Затухання сигналу
   ├── Фазовий зсув
   └── Вихідна напруга на RX
```

### Побачити графіки
```
1. Натисніть "📈 Графіки"
2. З'явиться вікно з 3 графіками:
   ├── α(f) — затухання від частоти
   ├── β(f) — фаза від частоти
   └── Z₀(f) — імпеданс від частоти
3. Миш часто скидає масштаб автоматично
```

## ⚡ Поширені завдання

### Задача 1: Розрахувати затухання кабелю 30м
```
1. ⊡ Приклад
2. Клік на кабель
3. Виберіть "ШВВП 2×1.5", "copper"
4. Встановіть довжину 30м
5. ✓ Результат: α = 144.19 Нп/м × 30м = 4325.7 Нп
```

### Задача 2: Порівняти мідь і алюміній
```
1. ⊡ Приклад
2. Клік на кабель → "copper"
   → Запишіть α₁ = 144.19
3. Клік на кабель → "aluminium"  
   → Запишіть α₂ = 238.53
4. ✓ Різниця: α₂/α₁ = 1.65× гірше
```

### Задача 3: Зберегти мою схему
```
1. Побудуйте схему (додайте вузли, кабелі)
2. 💾 Схема → виберіть імя "my_network.json"
3. ✓ Схема збережена
4. Завтра: 📂 Відкрити → "my_network.json"
5. ✓ Все повернулось як було!
```

## 🆘 Допомога при проблемах

### "Параметри не розраховуються"
```
❌ Вибрали вузол замість кабелю
✓ Рішення: Клікніть на ЛІНІЮ (кабель), не на точку
```

### "Не бачу комбобоксу типів кабелів"
```
❌ Правa панель не оновилась
✓ Рішення: Клікніть на кабель ще раз
```

### "Не можу видалити елемент"
```
❌ Елемент не вибраний
✓ Рішення: Клікніть на елемент, потім натисніть "🗑 Видалити"
```

### "Схема не зберігається"
```
❌ Не натиснули кнопку "💾 Схема"
✓ Рішення: 
   1. 💾 Схема
   2. Виберіть директорію
   3. Введіть ім'я файлу (наприклад: my_network.json)
   4. Натисніть "Зберегти"
```
