## 🆘 ПРОБЛЕМЫ & РЕШЕНИЯ

| Проблема | Решение |
|----------|---------|
| Параметры не рассчитываются | Кликните на **кабель** (линию), не на узел |

| Не сохраняется | Нажимите "💾 Схема", выберите файл |
| Не загружается | Выберите правильный JSON файл |

### CABLE_TYPES (в коде) vs cable_params.json (на диске)

- **cable_params.json** = результаты расчётов (α, β, Z₀)

👉 Разница объяснена в: **CABLE_TYPES_vs_JSON.md**
