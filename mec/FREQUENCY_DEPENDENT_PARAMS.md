# Частотно-залежні коефіцієнти α і β — Впровадження

## ✅ Виконано

Система тепер автоматично розраховує коефіцієнти затухання α і дисперсії β для діапазону частот від **0.01 до 1000 МГц** при виборі типу кабелю.

## 🔧 Що змінилося

### 1. **CableCalculator** — Нові методи

```python
# Розраховує спектр α, β, Z₀ для діапазону частот
calculate_params_spectrum(cable_type, material, freq_min_mhz=0.01, freq_max_mhz=1000, n_points=200)

# Інтерполює параметри на довільній частоті
get_params_at_freq(cable_type, material, frequency_mhz)
```

### 2. **Зберігання в JSON** — `cable_params.json`

Кожен тип кабелю з матеріалом тепер зберігає повний спектр:
```json
{
  "ШВВП 2×1.5_copper_spectrum": {
    "cable_type": "ШВВП 2×1.5",
    "material": "copper",
    "freq_mhz": [0.01, 0.017, ..., 1000],
    "alpha": [1.23, 2.15, ..., 5234.1],
    "beta": [4.56, 7.89, ..., 34567],
    "Z0": [66.8, 66.8, ..., 66.8]
  }
}
```

### 3. **Розрахунок каскаду** — `build_cascade()`

При розрахунку для кожної частоти f:
- Отримує інтерпольовані α(f) і β(f) з збереженого спектра
- Використовує їх як `γ = α + jβ` для ABCD-матриці
- Результат: точні розрахунки без статичних наближень

### 4. **Автоматичний запуск**

При виборі типу кабелю в інтерфейсі:
1. Одразу розраховується спектр (0.01–1000 МГц, 200 точок)
2. Зберігається у `cable_params.json`
3. При розрахунку схеми використовується
